# 💰 Expense Tracker (Streamlit)

A simple and interactive web application to track and manage daily expenses efficiently.

---

## 📌 Description

This Expense Tracker helps users record their daily spending, categorize expenses, and visualize spending patterns using charts. It is built using Streamlit for an easy-to-use interface.

---

## 🚀 Features

* ➕ Add daily expenses
* 📂 Select category (Food, Travel, Shopping, etc.)
* 📋 View all records in a table
* 📊 Visualize expenses using a pie chart
* ⚡ Simple and user-friendly interface

---

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
